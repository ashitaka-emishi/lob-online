# Specification: Edge Tool Panels Bundle — Road, Stream/Stone Wall, Contour Line, Ford & Bridge

**Track ID:** edge-tools-bundle_20260319
**Type:** Feature

## Summary

Implement the three remaining edge tool panels (Road, Stream & Stone Wall, Contour Line) plus
ford/bridge sub-controls, and complete the elevation gradient overlay deferred from PR #150.
Together these give the engineer a fully usable map digitization toolset for encoding the South
Mountain map before moving to UI and rules implementation.

## Context

Phase 1 scaffold is complete. The tool-panel framework (PR #150) delivered BaseToolPanel,
shared composables (useEdgePaintTool, useClickHexside, usePaintStroke), the declarative
overlayConfig system, and the edge data model (edge-model.js). The elevation and terrain panels
use BaseToolPanel but the elevation panel's gradient tint fill is explicitly deferred to this
track (#137 comment in ElevationToolPanel.vue). formulas/elevation.js does not yet exist.
All five new panels have detailed specifications in docs/designs/map-editor-design.md §6–§11.

## User Story

As an engineer digitizing the South Mountain map, I want fully functional elevation, terrain,
road, stream/stone wall, and contour line editors with accurate visual overlays so that I can
encode enough map data to stabilize a portion of the map and begin UI and rules implementation.

## Acceptance Criteria

- [ ] Elevation tool shows a gradient tint overlay (light blue → green → tan → brown) keyed to
      integer elevation level; gradient generated by elevationTintPalette() in elevation.js.
- [ ] Terrain tool shows per-terrain color fill overlay with toggleable layer via BaseToolPanel.
- [ ] Road tool paints trail / road / pike edges with a 'through-hex' line overlay distinguishing
      all three types; includes bridge sub-control (requires road/trail/pike on edge).
- [ ] Stream & Stone Wall tool paints stream / stoneWall edges with an 'along-edge' line overlay;
      includes ford sub-control (requires stream on edge).
- [ ] Contour Line tool paints elevation / slope / extremeSlope / verticalSlope edges with
      'along-edge' overlay; includes toggleable elevation tint + label ('Elevation info');
      includes auto-detect button with confirmation.
- [ ] All five tools are wired into MapEditorView and the accordion interaction gate.
- [ ] All new components and formula functions have Vitest tests; CI gates pass.

## Technical Notes

  tintForLevel(level, palette), autoDetectContourType(levelA, levelB). See design doc §15.
- Elevation overlay: tint palette interpolates HSL from hsl(185,50%,65%) at level 0 through
  green, tan, to hsl(25,50%,30%) at max level.
- ElevationToolPanel.vue already uses BaseToolPanel; it needs its overlayConfig filled in
  (currently passes an empty config — deferred by PR #150 comment referencing issue #137).
- Ford/bridge are sub-controls inside StreamWallToolPanel and RoadToolPanel respectively,
  using useClickHexside. Low additional effort given composable is built.
- All overlay colors and styles come from feature-types.js — no hardcoded values in panels.
