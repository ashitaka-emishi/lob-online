Update `server/src/routes/games.js` to assign the creator as 'confederate' and update
the join endpoint to accept and validate an explicit `side` parameter.

### Tasks

- [ ] Task 1.1: Update `POST /api/v1/games` — change creator side from `'union'` to
      `'confederate'` in the `setPlayerSession` call and response body.
- [ ] Task 1.2: Update `POST /api/v1/games/:id/join` — read `side` from `req.body`; validate
      it is `'union'` or `'confederate'`; return 400 if missing/invalid; return 409 with

      otherwise proceed with current join logic and `setPlayerSession(req, id, side, sideToken)`.

      expect `side: 'confederate'`; add join tests for: valid `side: 'union'`, invalid body
