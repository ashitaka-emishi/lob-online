## Context

Issue #197 (track `counter-detect_20260325`) was closed with all 172 units/batteries and 48

have since been addressed: (1) CSA command hierarchies in `oob.json` were wrong at the time,
and (2) the script did not leverage filename conventions or counter color to narrow the match
pool before calling the vision API. The OOB Editor exists as a manual fallback for any
remaining low-confidence or null refs.

## Technical Notes

Filename side-detection priority:

1. Prefix `C` (followed by digit) → CSA front cut-out
2. Prefix `U` (followed by digit) → USA front cut-out
3. `CS1-Front_*` / `CS1-Back_*` → face from name; side via blue-channel dominance test

`counterRef` schema shape (already in place):

```json
{ "front": "filename.png", "frontConfidence": 0.92, "back": "filename.jpg", "backConfidence": 0.87 }
```
