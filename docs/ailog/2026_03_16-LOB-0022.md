## AI Plan

Create `docs/claude-ecosystem/` as a unified reference hub for the lob-online agent/skill
ecosystem, aimed at developers who are new to the project or using non-Claude AI tooling
(e.g., GitHub Copilot / Codex). The work is pure documentation — no source code changes.
Five new markdown files will be authored:

- `README.md` — hub entry point with a "this is lob-online-specific" disclaimer and TOC
- `agents.md` — per-agent reference (name, purpose, collaborators, skills, tools) sourced
  from the live `.claude/agents/*.md` frontmatter and `docs/agents/*/design.md` files
- `skills.md` — per-skill reference (name, purpose, owning agent, related skills, Claude
  Code invocation, equivalent manual bash steps) sourced from `.claude/commands/*.md`
  and `docs/architecture.md`
- `network-diagram.md` — Mermaid diagrams for agent→skill ownership, skill→skill
  dependencies, and the full SDLC sequence; drawing from existing `docs/architecture.md`
  diagrams and extending them
- `guardrails-and-logging.md` — explains HCPs, CI gates, rules-lawyer gate, ailog, and
  devlog; sourced from `docs/agents/*/design.md` and observation of the existing workflow

One existing file is modified: `.claude/README.md` Documentation table gets a link to the
new hub. No unit tests apply; verification is `/dev-build` passing.

## Implementation Notes

All five new files are pure authored documentation — no source code changes. Content was
sourced directly from `.claude/agents/*.md` frontmatter, `.claude/commands/*.md` skill files,
and `docs/architecture.md` Mermaid diagrams.

Key decisions:

- `skills.md` includes a manual bash equivalent for every skill so Codex/Copilot users have
  actionable steps without needing Claude Code installed
- `network-diagram.md` extends the existing `docs/architecture.md` diagrams with a fourth
  diagram (issue intake detail flowchart) that wasn't present in the original
- `guardrails-and-logging.md` is ordered by "blast radius" — HCPs first (most immediate
  protection), then CI gates, rules gate, and finally the logging systems
- Prettier reformatted table column alignment in 4 files on the format pass; committed as a
  separate chore commit to keep the diff clean
